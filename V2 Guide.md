# V2.0 가이드

### 회원 로그인 기능 추가
- 이름과 초기 비밀번호로 로그인
- `홈`화면에서 비밀번호 변경

### 금주 매치리스트에서 출석 
- `금주`에 매치가 있는 경기 확인 가능
- 매치 일자에 출석 후 `출석` 버튼 클릭
- `임원`에게 출석 알림 메세지 발송
- 출석 버튼 대신 `출석한 날짜 및 시간` 노출

### 매치 상세에서 출석 리스트 확인 가능 
- `출석`한 `인원`들의 정보 조회 가능

### 골 기록
- 아무나 `골 등록` 시 `임원`에게 골 알림 메세지 발송
